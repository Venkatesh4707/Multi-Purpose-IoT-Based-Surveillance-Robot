# Circuit Diagram

This folder contains the block diagram and circuit diagram of the Multi-Purpose IoT Based Surveillance Robot.

## Contents

### 1. Block Diagram
The block diagram illustrates the overall architecture of the system, including:
- ESP32 Microcontroller
- ESP32-CAM Module
- Ultrasonic Sensor
- MQ2 Smoke Sensor
- Flame Sensor
- Metal Detector Sensor
- GPS Module (NEO-6M)
- Motor Driver (L293D)
- DC Motors
- MQTT Communication
- User Monitoring Device

### 2. Circuit Diagram
The circuit diagram shows the detailed hardware connections between:
- ESP32 Controller
- ESP32-CAM
- GPS Module
- Ultrasonic Sensor
- MQ2 Smoke Sensor
- Flame Sensor
- Metal Detector Sensor
- Motor Driver
- Buzzer
- Status LEDs
- Power Supply Unit

## Purpose

These diagrams provide a clear understanding of the hardware architecture, signal flow, and component interfacing used in the surveillance robot system.
