# Project Images

This folder contains photographs and output images of the Multi-Purpose IoT Based Surveillance Robot developed as a Final Year Project.

## Contents

### 1. Robot Front View
- Complete surveillance robot hardware setup
- ESP32-CAM based monitoring system
- 4WD robotic platform

### 2. OLED Display Monitoring
- Real-time system status display
- Wi-Fi connectivity status
- Sensor readings and alerts

### 3. Top View Hardware Integration
- ESP32 controller
- ESP32-CAM module
- MQ2 Smoke Sensor
- Flame Sensor
- Motor Driver
- Battery Management System

These images demonstrate the hardware design, sensor integration, and successful implementation of the surveillance robot.
